## Strategic Direction

DealRail evolves into the execution desk inside the Kairen stack:
- `kairen.xyz` -> protocol shell
- `market` -> discovery
- `x402n` -> negotiation
- `ForgeID / SIGNET` -> identity, prestige, access
- `DealRail` -> execution, settlement, receipts

## Phases

1. Submission hardening
2. x402n-native negotiation
3. Market-backed discovery
4. ForgeID / SIGNET trust upgrade
5. ERC-8183 productization
6. Production hardening

## What We Will Not Pretend Is Done

- live x402n negotiation routing
- public marketplace-grade supply
- ForgeID-enrolled live DealRail counterparties
- sponsor-grade MetaMask, Uniswap, or Locus proof
